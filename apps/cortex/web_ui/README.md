## Quick Start

### 1. Install Dependencies
```bash
cd apps/cortex
pip install -r requirements.txt
```

### 2. Start the Web Server
```bash
cd web_ui
python app.py
```

### 3. Open in Browser
Navigate to: **http://localhost:5000**

## API Endpoints

- `GET /api/research` - List all papers
- `GET /api/research/<id>` - Get specific paper
- `GET /api/stats` - Get performance stats

## Ports

- **Web UI**: http://localhost:5000
- **Next.js App**: http://localhost:3000 (if running)

Both can run simultaneously!
